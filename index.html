多周期交易系统完整规则

本系统包含三个阶段：纯均线交易 → 均线+力度K过渡 → 纯裸K交易

第一阶段：纯均线多周期交易系统

📊 基础设置

参数设置
均线类型EMA（指数移动平均线）
均线参数5（黄线）、13（粉线）、55（绿线）
进场周期5分钟（可选3/10/15分钟，需保持倍数关系）
过滤周期15分钟 + 30分钟

🔻 第一层过滤：30分钟周期（方向确认）

  • 做多条件：黄线(5) > 粉线(13) → 多头排列
  • 做空条件：黄线(5) < 粉线(13) → 空头排列
⚠️ 30分钟只看均线排列状态，确定大方向

🔻 第二层过滤：15分钟周期（双重确认）

做多条件（必须同时满足）：

  1. 均线黄粉金叉：黄线(5) 上穿 粉线(13) 或 黄线在粉线上方
  2. MACD金叉：MACD快线(DIF) 上穿 慢线(DEA) 或 DIF在DEA上方

做空条件（必须同时满足）：

  1. 均线黄粉死叉：黄线(5) 下穿 粉线(13) 或 黄线在粉线下方
  2. MACD死叉：MACD快线(DIF) 下穿 慢线(DEA) 或 DIF在DEA下方

第二阶段：均线 + 力度K 过渡系统

🔑 核心变化：找到力度K后，即使15/30分钟条件不完全满足也可以进场。

🔻 力度K线识别

![图2-力度K定义](2.jpeg)

图示说明：博弈区（阴阳交错）后出现的决定性大K线即为力度K。
特征说明
博弈区识别阴阳线叠加在一起的区域（多空争夺）
力度K确认博弈区后出现实体大于平均ATR的大阴/大阳线
有效性⚠️ 只有在博弈后出现的力度K才算数

🔻 过渡阶段进场方法

![图3-均线过渡到裸K](3.jpeg)

进场逻辑：在大周期（15/30min）确定力度K方向，在小周期（5min）按均线规则寻找触碰绿线后的突破信号。此阶段止损设在力度K的高/低点。

第三阶段：纯裸K交易系统

🔑 核心变化：删除均线，专注于裸K形态、关键位置（共振点）和力度K。

🔻 纯裸K进场方法（5分钟）

![图4-纯裸K进场](4.jpeg)

条件说明
多周期力度K共振15分钟和5分钟同时出现力度K
共振点识别价格回调到左侧高低点同一水平位置（支撑/压力）
进场信号共振点出现反向K线即可进场
止损设置力度K高点上方（空）或低点下方（多）

📌 三阶段进化总结

阶段 特点 大周期作用 进场依据
纯均线 严格过滤 均线排列+MACD 触碰绿线，突破粉线
均线+力度K 条件放宽 力度K定方向 均线信号
纯裸K 极简/专注价格 力度K+支撑压力 共振点+反向K线

🎯 核心要点总结

  • 博弈背景：力度K必须在博弈区后出现才具有指导意义。
  • 共振为王：多周期方向一致（共振）可极大提高胜率。
  • 位势一致：寻找水平共振点作为精准打击位。
  • 循序渐进：建议先熟练均线系统，产生盘感后再移除指标。
